Overview

Fours™ Classic is a minimalist 54-card deck designed to be played in four different ways.

From classic card games to fast duels, pattern matching, and grid strategy, each mode offers a distinct way to play — all using the same beautifully simple deck.

Built in black and white for clarity and accessibility, Fours™ Classic is easy to learn, quick to play, and endlessly replayable.

Game Modes

Game A

Classic Poker

A fully playable 54-card deck for traditional card games.

Game B

Rock • Paper • Scissors

A fast-paced duel mode built around the classic three symbol showdown.

Game C

Shape Matcher

A visual pattern matching game based on shared shapes and internal designs.

Game D

Tic-Tac-Toe

A competitive grid battle played with X and O cards on the included board.
